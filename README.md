# 🦁 Lion AI Multipurpose App

Welcome to the Lion AI Multipurpose App, a comprehensive Streamlit application designed to empower users with a suite of AI-driven tools. From intelligent research assistance to creative content generation, this app combines various functionalities into a single, intuitive platform.

## Features:

1.  **Research Assistant**: Engage in dynamic conversations with an AI assistant powered by Groq's Llama model. This feature provides fast and accurate answers to your questions, making research efficient and insightful.

2.  **StudyMate**: Your personal research and homework assistant. Upload your notes (PDF or TXT), and StudyMate will index them, allowing you to ask questions directly about your uploaded material. It can also perform web searches if the answer isn't in your notes, and includes a calculator tool for mathematical expressions.

3.  **Convert Voice To Text**: Transcribe audio files (WAV, MP3) into text using advanced Automatic Speech Recognition (ASR) technology. Ideal for converting lectures, interviews, or meetings into searchable text.

4.  **Convert Text To PDF**: Easily transform any text input into a downloadable PDF document. Perfect for creating reports, notes, or any text-based document.

5.  **Generate Prompt For Image**: Unleash your creativity by describing your desired image, and the AI will craft a detailed and artistic prompt optimized for modern image generation models like DALL-E or Midjourney.

6.  **Generate Image With Lion**: Bring your text descriptions to life by generating images directly within the app using state-of-the-art Stable Diffusion models. (Requires a Hugging Face token).

## Authentication:

The application includes a secure user authentication system with sign-up and sign-in functionalities, ensuring personalized experiences and data privacy.

## Technologies Used:

*   **Streamlit**: For creating interactive web applications.
*   **Groq API**: Powering the conversational AI and tool-calling capabilities.
*   **Tavily API**: For intelligent web search functionalities.
*   **ChromaDB & Sentence Transformers**: For efficient document indexing and retrieval in StudyMate.
*   **Hugging Face Transformers & Diffusers**: For advanced ASR (Whisper) and Stable Diffusion image generation.
*   **PyPDF, gTTS, MoviePy, SpeechRecognition, FPDF, SQLite3, bcrypt, pydub**: Supporting various multimedia, PDF, database, and utility functions.

## Setup and Usage:

Detailed instructions for setting up environment variables (GROQ_API_KEY, TAVILY_API_KEY, HF_TOKEN) and running the application can be found within the notebook. Ensure you have the necessary API keys configured to unlock all functionalities.

This app is designed to be a versatile tool for students, researchers, content creators, and anyone looking to leverage AI in their daily tasks.
